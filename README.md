# 🌾 Farmer's Help Bot

An AI-powered agriculture assistant designed to support farmers with instant guidance, crop-related information, farming tips, and smart assistance using modern technology.

---

## 🚀 Features

* 🌱 Crop guidance and recommendations
* 🌦️ Weather-related farming support
* 🐛 Pest and disease information
* 🤖 AI chatbot interaction
* 📄 Easy-to-use interface
* 📊 Smart farming assistance
* 🌍 Supports digital agriculture initiatives

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/muskan-dhawan/Farmer-s-Help-Bot.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd Farmer-s-Help-Bot
```

### 3️⃣ Create Virtual Environment

```bash
python -m venv venv
```

### 4️⃣ Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Mac/Linux

```bash
source venv/bin/activate
```

### 5️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
python bot.py
```

The application will start on:

```bash
http://127.0.0.1:5000
```
